Bot 2: self play

The idea is to train the AI using games it plays against itself. After a game has been played, the board state before each of the winners moves is set as input to train on, with the choosen move as the correct output. When the AI plays a draw against itself, it instead plays a game against a random opponent.

Bot 1: completing moves

In order to train the AI a database of games is created by letting two random bots play as many games as needed, keeping only those with a desicive winner. The board states before the winning moves are then of set as input to train on, with the winning move as the correct output.

Tic-tac-toe

The XOR problem

The XOR-problem is the most basic problem to solve with a neural network. It is here solved with small network (press Show network for an illustration), using supervised learning. The latter means that the network output is calculated for given a certain input, which is then compared to a desired output. The weights are then changed so that the next time the same input is used, the output will be closer to the correct answer.
